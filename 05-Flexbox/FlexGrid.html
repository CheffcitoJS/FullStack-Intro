Introducción a Flexbox y Grid

Flexbox y Grid son dos sistemas de diseño en CSS que permiten crear layouts modernos y responsivos de manera eficiente. Ambos tienen sus fortalezas y son esenciales para cualquier desarrollador web.

  • Flexbox: Ideal para diseños unidimensionales (en fila o columna).
  • Grid: Ideal para diseños bidimensionales (en filas y columnas).

Flexbox

Flexbox (Flexible Box) es un sistema de diseño en CSS que organiza elementos dentro de un contenedor. Su principal ventaja es la alineación y distribución eficiente de elementos, incluso cuando su tamaño es dinámico.

Ejemplo de Código:

/* Contenedor */
.container {
    display: flex;
    justify-content: center;
    align-items: center;
    height: 200px;
    background-color: #ff8c42;
}

/* Elementos */
.item {
    width: 50px;
    height: 50px;
    background-color: #292929;
    margin: 5px;
}

Grid

CSS Grid es un sistema de diseño bidimensional que organiza elementos en filas y columnas. Ofrece una manera precisa y poderosa de crear layouts complejos.

Ejemplo de Código:

/* Contenedor */
.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 10px;
    background-color: #ff8c42;
}

/* Elementos */
.grid-item {
    height: 100px;
    background-color: #292929;
}
