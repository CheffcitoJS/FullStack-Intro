Introducción a los Formularios en HTML

Los formularios en HTML son una herramienta fundamental para la interacción entre usuarios y aplicaciones web. Permiten recopilar datos, realizar búsquedas, iniciar sesión, entre otras funcionalidades clave. Con Bootstrap, puedes estilizar y organizar formularios de manera profesional y responsiva.

Tipos de Formularios

Campo de Texto

Permite a los usuarios ingresar texto libre. Utiliza el elemento <input type="text">.

<input type="text" class="form-control" placeholder="Escribe aquí">
Contraseña

Se utiliza para ingresar contraseñas, ocultando el texto ingresado. Utiliza <input type="password">.

<input type="password" class="form-control" placeholder="Contraseña">
Checkbox

Permite a los usuarios seleccionar múltiples opciones. Utiliza <input type="checkbox">.

<input type="checkbox" class="form-check-input"> Opción 1
Radio Button

Permite seleccionar una única opción entre varias. Utiliza <input type="radio">.

<input type="radio" class="form-check-input" name="grupo1"> Opción 1
Área de Texto

Se utiliza para ingresar textos largos. Utiliza el elemento <textarea>.

<textarea class="form-control" rows="3" placeholder="Escribe aquí"></textarea>
Select

Permite seleccionar una opción de un menú desplegable. Utiliza <select>.

<select class="form-select">
  <option value="1">Opción 1</option>
  <option value="2">Opción 2</option>
</select>
